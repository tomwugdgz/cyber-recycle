### 2️⃣ MCP 集成方案

**位置**: `cyber-recycle/mcp-integration/README.md`

**内容概要**:

#### 可用工具（7 个）
1. **collect_data** - 数据采集（Slack/Jira/GitHub/Confluence）
2. **mint_data_nft** - 铸造数据 NFT
3. **distill_dataset** - 数据蒸馏（L1-L5 等级）
4. **create_rl_environment** - 创建 RL 环境
5. **purchase_data** - 购买数据
6. **subscribe_rl_environment** - 订阅 RL 环境
7. **execute_rl_call** - 执行 RL 调用（A2A 支付）
8. **list_datasets** - 浏览数据市场

#### 配置示例
```json
{
  "mcpServers": {
    "cyber-recycle": {
      "command": "npx",
      "args": ["-y", "@cyber-recycle/mcp-server"],
      "env": {
        "API_KEY": "your_api_key"
      }
    }
  }
}
```

#### 使用示例
```python
# 采集 Slack 数据
data = await client.collect(
    channels=['general', 'engineering'],
    date_range=('2025-01-01', '2025-12-31'),
    sanitize=True
)

# 蒸馏成 RL 环境
rl_env = await pipeline.distill(data_cid, level=5)
```
